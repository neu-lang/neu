# Review: M0028-003

## Decision

`APPROVE`

## Checks

- Task scope matches M0028 and leaves calls, runtime behavior, and static
  arithmetic diagnostics untouched.
- M0018 core remains routed through the same shared logic without executable
  metadata; its regression tests pass.
- M0028 core types accepted operators before initializer and assignment checks,
  while `!` and `==` retain deferral diagnostics.
- Tests preceded implementation. Adversarial report and full CI pass.

## Spec Compliance

Compliant with ADR-0042 and ADR-0043: only the listed `Int` operators are
accepted by the new path. No new semantics are introduced.

## Residual Risk

The core is not yet connected to a full compiler driver; this remains explicit
M0028 scope for later tasks.
